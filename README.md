# Asif Barakat Chowdhury — Portfolio

A fast, animated single-page portfolio built with **React 19 + Vite + Tailwind CSS v4 + Framer Motion**.

🔗 **Live site:** _add your Vercel URL here after deploying_

---

## ✨ Highlights

- Systems & AI research (split computing, edge AI, distributed LLM inference)
- Featured software projects
- Competitive programming
- Fully responsive, dark-themed, animated

## 🛠 Tech Stack

| Tool | Purpose |
|---|---|
| Vite 7 | Build tool & dev server |
| React 19 | UI library |
| Tailwind CSS v4 | Styling (utility classes) |
| Framer Motion | Animations |
| react-icons | Icons |

## 🚀 Getting Started

```bash
npm install      # install dependencies
npm run dev      # start dev server → http://localhost:5173
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## ✏️ Editing your content

All text, skills, projects, and research live in **one file**:

```
src/data/portfolio.js
```

Edit the values there and the whole site updates — no component code needs to change.

## ☁️ Deploy

This site deploys to [Vercel](https://vercel.com) or [Netlify](https://netlify.com)
with zero config — the framework is auto-detected (build: `npm run build`, output: `dist`).
